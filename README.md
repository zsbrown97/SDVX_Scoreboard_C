# SDVX Scorecard
An attempt at making something in C involving my current hyperfixation: Sound Voltex

I've started recording my Sound Voltex gameplay and saving the files in the following format:

`SDVX_Song Name_Difficulty_Score_Rating Date Time.mkv`

`SDVX_Bad Apple!!_EXH12_9873358_AAA+ 03-09-26 11-48-26.mkv`

More examples are in the `sampledata` directory as blank text files.

I want to parse each file name and print the individual values to a table, as a record of songs I've played, and my scores of each playthrough.
